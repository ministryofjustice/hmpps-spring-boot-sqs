# 6.0.0

Upgrade to spring-boot [4.0.1](https://github.com/spring-projects/spring-boot/releases/tag/v4.0.1)

See https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-4.0-Migration-Guide for the migration guide.

## Upgrade to Kotlin 2.3.0

This includes an [upgrade to Kotlin 2.3.0](https://github.com/JetBrains/kotlin/releases/tag/v2.3.0)

Spring Boot 4.0.0 has moved to Jackson 3, however, the AWS SDK still uses Jackson 2 so we have added
```kotlin
  api("org.springframework.boot:spring-boot-jackson2")
```
to the auto-configuration.

## Build plugin version updates

- org.springframework.boot:org.springframework.boot.gradle.plugin [3.5.8 -> 4.0.1]
